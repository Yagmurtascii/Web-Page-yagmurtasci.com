Raspberry Pi

Raspberry Pi Kurulumu

Eğer raspberry pi eski bir model varsa elinizde; NOOBS, Raspberry Pi Os, Raspberry Pi OS Lite, NOOBS Lite kurulumlarını nasıl yaparız onları göreceğiz. İhtiyacımız olan malzemeler;

  • Raspberry pi herhangi bir model
  • Güç kablosu
  • İnternet kablosu
  • HDMI bağlantısı
  • USB klavye ve Mouse
  • Win32 Disk İmager
  • SD Card Formatter
  • SD kart okuyucu
  • SD kart

SD kart modelinizi raspberry pi’ye uygun seçilmelidir. Class 10 olması önerilir. Başlangıç için en az 8GB lık bir SD karta sahip olmanız şarttır. Ama Raspberry pi modeliniz eskiyse 16-32 GB lık SD kartlar seçebilirsiniz. Kurulduktan sonra yavaş çalışabiliyor.

Kurulum

NOT: Raspberry Pi Os, Raspberry Pi Os Lite gibi işletim sistemlerini doğrudan SD karta yazdırabilirsiniz. Veya NOOBS üzerinden kurulum gerçekleştirebilirsiniz.


Raspberry Pi Os ve Raspberry Pi Os Lite(Masaüstü desteği mevcut değil) işletim sistemlerini kurmak için;

  • SD kart okuyucumuzu bilgisayara takıyoruz ve bizim için uygun olan SD kart formatına kartımızı yerleştiriyoruz.
  • SD Card Formatter programını açıp SD kartımızı formatlıyoruz. SD kartınızı formatladığınızdan emin olun.
  • Raspberry Pi sitesinden Zip şeklinde ya da Torrent üzerinden indirin.

  • Sağ tuş yapıp “Tümünü Ayıkla” kısmını seçip içindeki disk görüntüleyiciyi buluyoruz.
  • Win32 Disk Imager programını açıp disk görüntü dosyasını seçiyoruz. Device kısmını doğru seçtiğinizden emin olun.
  • Yükleme bittikten sonra SD kartımızı sağ tuşa basıp güvenli bir şekilde çıkarıyoruz.
  • Raspberry Pi Os veya Raspberry Pi Os Lite işletim sistemi kurulu SD kartımızı Raspberry Pi ‘ye yerleştiriyoruz.
  • USB klavye,Mouse,güç kablosu,hdmi kablosu,internet kablosunu taktıktan sonra Rasppberry pi’ye güç veriyoruz.

Bu ekrandan açıldıktan sonra kapanıyorsa yapmanız gerekenler;

  • Raspberry Pi gücümüzü kesiyoruz ve SD kartımızı Raspberry Pi ‘den çıkarıyoruz. SD kart okuyucumuzu bilgisayara takıyoruz ve bizim için uygun olan SD kart formatına kartımızı yerleştiriyoruz.
  • Bundan sonra yapılması gereken asıl işlem SD kart içinde bulunan config.txt dosyasında birkaç düzenleme yapmak.
  • Config.txt dosyanı açtığımızda böyle bir ekranla karşılaşırız.
  • Düzenleyeceğimiz verilerin önünde yer alan # sembolü kaldırarak yorum satırlarını etkisizleştiriyoruz.
  • Değiştirmemiz gereken değerler;
    • hdmi_mode=4
    • hdmi_group=2
    • hdmi_force_hotplug=1
    • disable_overscan=0
    • overscan_right=24
    • overscan_left=24
    • overscan_top=24
    • overscan_bottom=24
    • config_hdmi_boost=4
    • hdmi_ignore_edid=0xa5000080 (Eğer bu ifade config.txt içinde bulamadıysanız herhangi bir yere ekleyebilirsiniz.)
  • Düzenleme bittikten sonra SD kartımızı sağ tuşa basıp güvenli bir şekilde çıkarıyoruz.
  • Raspberry Pi Os veya Raspberry Pi Os Lite işletim sistemi kurulu SD kartımızı Raspberry Pi ‘ye yerleştiriyoruz.
  • USB klavye,Mouse,güç kablosu,hdmi kablosu,internet kablosunu taktıktan sonra Rasppberry pi’ye güç veriyoruz.

Ve işte oldu. Raspberry Pi kullanıma hazır.
